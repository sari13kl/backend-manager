## Visão Geral

### Gerenciador de Tarefas

Irá funcionar em torno dos seguintes pilares estratégicos:

- **Usuários**
  - Gestão de contas
  - Base para personalização e segurança

- **Tarefas**
  - Criação, edição e exclusão
  - O ativo principal do sistema

- **Status**
  - 'Pendente'
  - 'Em progresso'
  - 'Concluída'

- **Datas**
  - Data de criação
  - Prazo (deadline)

- **Prioridade**
  - Baixa / Média / Alta  
  - Nem toda tarefa merece atenção imediata

- **Autenticação**
  - Login seguro
  - Controle de acesso

### Evolução

Funcionalidades pensadas para escalar o valor do produto ao longo do tempo:

- Categorias
- Lembretes
- Notificações

> Estratégia: começar simples, validar uso real e evoluir com inteligência.
